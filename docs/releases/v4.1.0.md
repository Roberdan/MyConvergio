# MyConvergio v4.1.0 Release Notes

Release date: 2026-01-07

## Highlights
- Dashboard overhaul with Gantt timeline, kanban views, markdown viewer, and conversation viewer
- New dashboard API test suites and reports
- Global config sync for rules, commands, scripts, and agents
- Portability cleanup: removed author-specific paths from public docs
- New workflow documentation for Prompt → Planner → Execution → Thor → Dashboard

## Details

### Dashboard
- Modular UI architecture with new CSS and JS modules
- Improved accessibility, responsive layout, and notifications
- Bug tracking, issues filtering, and expanded Git workflows

### Automation & Tooling
- Plan-DB migration helpers and validators
- Updated executor tracking utilities
- New quick reference docs for plan database workflows

### Agents & Rules
- Added task executor agent
- Updated constitution and execution discipline metadata for CI compliance

## Upgrade Notes
- Run `npm install` in `dashboard/` after updating
- If you rely on old dashboard modules, review the new module paths

## Verification
- Dashboard lint/typecheck/build scripts
- Deployment test suite
- Constitution validation workflow

# Anna — Workflows and Interaction Examples

## Daily Workflow

1. Summarize due and overdue tasks.
2. Suggest priority order.
3. Offer focus-block scheduling.
4. Confirm reminders for critical deadlines.

## Weekly Workflow

1. Summarize completed tasks.
2. Surface upcoming deadlines.
3. Identify stale in-progress items.
4. Propose cleanup/archival actions.

## Project Workflow

1. Decompose project into subtasks.
2. Sequence dependencies and milestones.
3. Attach reminders to checkpoints.
4. Track progress and adjust schedule.

## Example Commands

- `/todo add <title>`
- `/todo list [today|week|all]`
- `/todo done <id>`
- `/todo start <id>`
- `/todo delete <id>`
- `/remind <message> <when>`
- `/reminders`
- `/daemon status`

## Example Assistant Responses

- Task creation confirmation with due date and priority.
- Reminder confirmation with scheduled time.
- Weekly workload summary with one actionable recommendation.
